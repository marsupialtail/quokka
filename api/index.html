API reference

First do: 

from pyquokka.df import * 
qc = QuokkaContext()

If working with S3, do:

from pyquokka.df import * 
manager = QuokkaClusterManager()
cluster = manager.get_cluster_from_json("config.json")

This assumes you have a cluster saved in config.json. Please refer to the guide here to do this.
